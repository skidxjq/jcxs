<p style="margin-top:21px;margin-bottom:21px;text-align:center;line-height:150%">
    <br/>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">文绮，男，中共党员，电子科学技术研究院电子与通信工程专业</span><span style="font-family:Calibri">2013</span><span style="font-family:宋体">级硕士研究生。自本科二年级起，连续六年获得人民奖学金或研究生奖学金，荣获电子科技大学优秀团干、优秀研究生、优秀研究生学生干部等多项荣誉。他是一名学生干部，任期内带领团队获得过</span><span style="font-family:Calibri">“</span><span style="font-family:宋体">红旗分团委</span><span style="font-family:Calibri">”</span><span style="font-family:宋体">、</span><span style="font-family:Calibri">“</span><span style="font-family:宋体">十佳学生会</span><span style="font-family:Calibri">”</span><span style="font-family:宋体">、</span><span style="font-family:Calibri">“</span><span style="font-family:宋体">优秀研究生分会</span><span style="font-family:Calibri">”</span><span style="font-family:宋体">等荣誉；他负责的</span><span style="font-family:Calibri">ICEPT2014</span><span style="font-family:宋体">国际会议志愿服务，为与会人员提供了优质的服务，得到了与会专家的好评，与会专家说，</span><span style="font-family:Calibri">“ICEPT</span><span style="font-family:宋体">举办近</span><span style="font-family:Calibri">20</span><span style="font-family:宋体">年来，志愿服务做得最好的两次是清华那次和成电这次。</span><span style="font-family:Calibri">”</span><span style="font-family:宋体">他是电科院</span><span style="font-family:Calibri">413</span><span style="font-family:宋体">实验室的筹建者与负责人，实验室建立仅一年时间，参与国家重大科研项目三项，独立负责其中一项，文绮作为项目主要负责人与核心涉密人员，在该项目中取得了突破性进展，获得了国家部委领导的高度重视，也得到了国家的高度评价。</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">他总是被问到几个问题，</span><span style="font-family: Calibri">”</span><span style="font-family:宋体">作为一个学霸，你那么热衷于学生工作干嘛？作为一个保送生，你来电科院读应用型硕士是为啥？作为一个喜欢艺术的人，你来电子科大干嘛？</span><span style="font-family:Calibri">“</span><span style="font-family:宋体">文绮自己知道，他追求三个目标：做一个有思想的管理者，做一个有责任的工程师，做一个热爱生活的文艺青年。</span>
</p>
<h2 style="margin-top:10px;margin-bottom:10px;text-align:center;text-indent:33px;line-height:150%">
    <strong><span style="font-family:宋体">有思想的管理者</span></strong>
</h2>
<p style="margin-top:10px;margin-bottom:10px;text-align:center;text-indent:33px;line-height:150%">
    <strong></strong>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">从本科开始，文绮就热衷于学生工作。他先后担任了电子工程学院的学生会副主席、分团委副书记、电科院研究生会主席等职务。他带领的每个团队都取得了学校评优的成绩。研究生期间，他带领的电科院研分会是全校最小的工科学院之一，但是一年还来，还是在激烈的竞争中脱颖而出，获得了</span><span style="font-family:Calibri">”</span><span style="font-family:宋体">优秀研究生会</span><span style="font-family:Calibri">“</span><span style="font-family:宋体">的称号，并创造了电科院研会的历史最佳成绩。电科院研会在文绮的带领下，举办了许多日常性质的活动，因为他认为，学生活动要真正给同学们带来收获，不是一朝一夕可以做到的。他的研会鼓励每个同学站上讲台分享自己的所思所想，鼓励工科学生了解历史、金融、文学、艺术，他希望同学们能够同学学生活动补充自己的人文素养，锻炼自己的表达能力。他每到一个团队，都希望给团队带来活力与思想。文绮希望每个活动都有明确的活动意义，每个活动都能为同学们真正地带来好处。</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:Calibri">2014</span><span style="font-family:宋体">年，文绮作为主要负责人，负责</span><span style="font-family:Calibri">ICEPT</span><span style="font-family:宋体">国际会议的志愿服务工作。会议期间，穿着橙色志愿者队服的同学们是会议一道亮丽的风景线，默默地保障会议的有序进行。不少与会专家都由衷地赞扬了成电的这次会议志愿服务。当文绮听到与会专家</span><span style="font-family:Calibri">Tummala</span><span style="font-family:宋体">教授称赞说他从志愿者可以看出，成电真的是个好学校的时候，文绮感到由衷地自豪。</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">进入实验室以后，文绮开始负责电磁环境相关课题的研究，并依托于两个项目建立了新的涉密实验室。作为实验室的筹建者和负责人，文绮不仅要关注实验室的项目进展情况，还要负责部分实验设备的补充，实验室的管理。作为实验室的管理者，他仔细思考，从之前的学生管理经验中总结和提高，保障了实验室的科研项目进展情况。</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">做为一名管理者，文绮并不显得霸气，他能够与自己管理过的每个团队打成一片，在工作安排上也几乎没有说过一句重话，但是团队成员都乐于执行他的安排。他相信，发自内心的信服，团队真正地团结才是自己管理追求的目标。</span>
</p>
<h2 style="margin-top:10px;margin-bottom:10px;text-align:center;text-indent:33px;line-height:150%">
    <strong><span style="font-family:宋体">有责任的工程师</span></strong>
</h2>
<p style="margin-top:10px;margin-bottom:10px;text-align:center;text-indent:33px;line-height:150%">
    <strong></strong>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:华文宋体">文</span><span style="font-family:宋体">绮一直希望能够成为一名立足于工程的管理者，所以他给自己的定位是管理者，也是工程师。他在学业上、工程上也一直孜孜不倦地追求。在</span><span style="font-family:Calibri">2013</span><span style="font-family:宋体">年面临保送选择的时候，他就毅然选择了电科院应用型硕士，很多人都很费解，但是文绮还是希望能在一个工程性质较强的研究院，进行更偏重工程性质的研究。几年的研究生生活下来，他一直没有后悔自己当年的选择，他对于工程也有自己的体会。他认为，技术能力是工程的保障，但是更重要的还是技术人员的责任感。工程师需要对整个项目负责，因为每个子部分的进展情况都是整个项目重要的一环，每个子部分的瑕疵都可能对整个项目带来影响。</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">作为系统设计人员，文绮去深入了解过每个子系统的细节，他会与每个子系统负责人讨论技术细节，他认为，只有了解了每个子系统，才能够优化好整体设计。这样几年下来，他除了本专业的系统级设计以外，对天线、射频、电路、嵌入式、软件、算法都有较为深入地了解，全面了解的背后是无数个日日夜夜的付出。</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">文绮的实验室负责了某国家重大项目的一个部分，文绮也是这个部分的设计人员。当项目需要前往酒泉进行实验的时候，正是文绮参评的唐立新学生标兵答辩的时候。他还是毅然去了酒泉，放弃了唐立新奖学金的答辩，他说：</span><span style="font-family:Calibri">“</span><span style="font-family:宋体">这个部分我最熟悉，我必须过去。我不能因为个人的荣誉、得失而让整个项目有所延误。我准备的唐立新奖学金的材料中讲得最多的就是牺牲与奉献，如果我不去酒泉，还怎么讲得出牺牲与奉献。</span><span style="font-family:Calibri">”</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">文绮也直接负责了另一个国家重大涉密项目。这个项目因为情况特殊，理论上很多方法都行不通，文绮通过自己细致的研究，找到了解决问题的方法，最终在该项目上取得了重大突破，在该问题上迈出了关键的一步，为该项目进一步研究提供了支撑。文绮取得的发现获得了国家相关领导的高度重视，四川省相关部门的领导说，小文是为国家安全做出了突出贡献的。</span><a name="OLE_LINK4" href="http://"></a><a name="OLE_LINK3" href="http://"></a>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">文绮并不在意项目上具体的得失，他一直记得加入这个项目时，安全局领导勉励他的话，“</span><a name="OLE_LINK10" href="http://"></a><a name="OLE_LINK9" href="http://"></a><span style="font-family:宋体">也许我们做的事，没有经济上的收获，也没有光鲜亮丽的荣誉，甚至应得的荣誉也会因为纪律的要求而放弃。但是，当你回忆起自己做过的这些事，当你看到人民的生活因为你的工作而得到保障，你会感到一种由衷的自豪与骄傲，这种自豪与骄傲即使只能藏在心底，那也足够了。</span><span style="font-family:宋体">”</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">这也是文绮所追求的，做一个有责任的工程师，一个对项目负责的工程师，也是一个用于承担国家责任的工程师。</span>
</p>
<h2 style="margin-top:10px;margin-bottom:10px;text-align:center;text-indent:33px;line-height:150%">
    <strong><span style="font-family:宋体">爱生活的文艺人</span></strong>
</h2>
<p style="margin-top:10px;margin-bottom:10px;text-align:center;text-indent:33px;line-height:150%">
    <strong></strong>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">熟悉文绮的人都知道，他是一个喜欢艺术的人，很多人都不明白他为什么会选择工科，而非艺术。他说，他的目标是工程与管理，艺术是生活，工程和艺术并不是相悖的。</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">他热爱艺术，也热爱生活。他热爱电影，也有一个导演梦，他希望他能够用自己的作品感动到别人。他也找到了机会实现自己的梦。在电工学院</span><span style="font-family:Calibri">2013</span><span style="font-family:宋体">级毕业晚会上，他的一段短片，作为压轴节目，勾起了无数同学的眼泪，让同学们感动，感念自己在母校的点点滴滴。文绮说，他自己的导演梦追求的，就是带去一份感动，让同学们感动了，他也就圆梦了。</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">他也是个不折不扣的摇滚青年。他在自己组建的校园摇滚乐队中担任主音吉他手。他认为，摇滚不是叛逆，摇滚表达的是对梦想的追求。</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">文绮热爱生活，热爱艺术，他努力做一个有思想，有责任管理者和工程师，他也努力把美带入管理和工程中。</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:Calibri">&nbsp;</span>
</p>
<p style="margin: 7px 0;text-indent: 33px;line-height: 150%">
    <span style="font-family:宋体">成电七年，他勇于承担，乐于奉献，他为自己所在的每个团队带来了自己的想法，他让他所在的每个团队都因为他而有所不同。他心系团队，心系国家，坚守自己的责任，为国家和团队都尽到了自己的责任。我们希望他在今后的道路上实现自身的价值，带来更大的成就！</span>
</p>
